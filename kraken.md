# Kraken

This is an economy build for the Kraken clan.

## Build Order

### Early game

- Scout Camp
- Fishery
- House
- Wood Cutter
- Wood Cutter
- Fishery
- Food Silo
- House
- Food Silo
- Horgr

### Mid Game

- Mine (Iron and Stone)
- Forge
  - Upgrade Fishermen, Norns, and Merchants
- Trading Post
  - Trade largest resource to Neutral
- House

### Late Game

- Upgrade Townhall
- Upgrade Fisheries
- Trading Post
- Warcamp
  - Get a Valkyrie
- Legion Military Path
- Trading Post
- Warcamps
  - Start looking at getting to a 15 or 18 warrior count
- Upgrade Warriors
- Upgrade Silos
- Upgrade Horgr
- Build up army
